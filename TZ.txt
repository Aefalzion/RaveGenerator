ТЗ:
Описание проекта: программа получает на ввод большой объём связного текста (например, "Войну и мир"). На основе этого текста программа собирает статистику того, с какой вероятностью после некоторой последовательности слов будет следовать определённое слово. Затем использую эту статистику она должна генерировать новый текст. Также должна быть возможность менять глубину поиска по статистике (длину последовательности для которой ищется вероятность следующего слова) без необходимости повторного её анализа.

Что нужно сделать:
- Получение статистики из файлов:
  - Функция, записывающая данные из файла в строку.
  - Функция, собирающая статистику по строке 
- Структуры хранения данных:
  - Структура хранящая все слова (пара “слово — id”) (Она же Словарь)
  - Структура дерева Маркова (здесь хранится статистика по последовательностям с использованием Id слов)
- Генерация нового текста:
  - Функция, генерирующая текст на основе дерева Маркова 
  - Функция, приводящая текст к привычному для человека виду
- Визуальное оформление:
  - Реализация удобного управления программой из командной строки

Правки/новые идеи:
- По факту реализована вспомогательная структура — дерево пар «число — указатель»
- Реализовать функцию печати словаря, составленного на основе книг
- Реализовать функцию поиска всех возможных продолжений заданной последовательности слов со статистикой частоты в дереве Маркова
- Сделать минимальное логирование для отладки
- Сделать презентацию проекта
